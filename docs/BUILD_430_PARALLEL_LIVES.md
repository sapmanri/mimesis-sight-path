# BUILD 430 — Parallel Lives

> Vase 지시 2026-07-18. 원문은 "BUILD 429"였으나 **429는 Genome Seasons가 선점** → 다음 번호 430으로 배정(번호 충돌 규칙).
>
> **같은 세상을 살지만 서로의 세상이 되지는 않는다.**
> 둘은 서로를 *인식*하지만 서로를 *주인공*으로 만들지 않는다. 별이의 세계도 빼콩의 세계도 주인공은 세상이다.
> 둘이 서로만 바라보기 시작하면 그 순간 "캐릭터 게임"이 된다.

## 0. 최상위 하드룰 (게이트가 강제 — `validate:parallel`)

1. **세계는 둘보다 크다.** 둘이 스쳐도 세상은 멈추지 않는다 — `update()`는 어떤 조기 return보다 먼저 `sky.tick`·`ppae.tick`을 부른다. 구름은 흐르고 새는 날고 날씨는 계속된다.
2. **서로를 기록하지 않는다.** 어떤 문장 풀에도 상대가 등장하지 않는다("빼콩을 만났다" ✗, "별이를 만났다" ✗). 항상 세상을 기록한다.
3. **스침엔 기록이 없다.** 스쳐 지나감 경로에서 일기·스트림 출력이 발생하면 게이트 실패.

검증기는 음성 테스트 3종으로 자기 검증됨(상대 지목 문장 주입 / `sky.tick` 뒤로 이동 / 스침에 기록 추가 → 전부 exit 1).

## 1. 430-A (배포됨)

| 지시서 | 구현 |
|---|---|
| ① 가끔 스쳐 지나간다 ⭐5 | 마주치면 **둘 다** 0.35~0.5s 멈춘다. 대화·기록 없음. 최소 20s 간격으로 드물게 |
| ② 서로를 피해간다 ⭐4 | 스침의 50%에서 별이는 한 발 옆(1px), 빼콩은 살짝 물러선다 |
| ③ 같은 풍경을 본다 ⭐5 | **427에 이미 있음** — `PpaeBrain.parallel`이 같은 eventId로 다른 시선 기록(30%) |
| ④ 흔적만 남는다 ⭐5 | `prop.trace` (TTL 2분): 별이가 쉰 자리 → `warm`, 빼콩이 앞발로 건드린 것 → `moved`. **남긴 주체는 문장에 등장하지 않는다** |
| ⑦ 같은 장소 다른 기억 ⭐5 | 흔적으로 구현 — 벤치: 별이 "잠깐 쉬었다" / 뒤이어 빼콩 "사람 냄새가 남아 있었다" |
| ⑥ 서로를 기록하지 않는다 ⭐5 | 게이트로 승격(위 하드룰 2) |
| 우연 1% | 사진 찍는 순간 빼콩이 프레임을 가로지른다(`PARALLEL.coincidence=0.01`). 캡처는 캔버스를 크롭하므로 **엽서·사진첩에 실제로 찍힌다.** 별이는 모르고, 보는 사람만 안다 |

문장 풀: `PPAE_TRACE_WARM`(온기·냄새) / `BYEOL_TRACE_MOVED`(자리가 달라짐) — 둘 다 주체 없는 관찰문.

## 2. 남은 단계 (430-B 후보)

- ⑤ 같은 비 — 별이 "비가 오래 내렸다" / 빼콩 "발이 젖었다": 빼콩 쪽은 427 `awayWet`으로 존재, 별이 쪽 날씨 일기는 미구현
- ⑧ 같은 시간 다른 행동 — 노을에 별이는 하늘/빼콩은 벌레. 현재 각자 Brain으로 자연 분기하나 **시간대 가중치는 없음**
- 우연 2종 — 풀숲에 가려 별이는 못 보고 사용자만 아는 순간(전경 오브젝트 뒤 배치 필요)
- 흔적 확장 — 물웅덩이 발자국, 눈 위 자국 등 날씨 결합
